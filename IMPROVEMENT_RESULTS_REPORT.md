## 🚀 **バッチ返信システム**

### **GitHub Pull Request Reviews API活用**
```bash
# 複数コメントを1回のAPIで送信
curl -X POST \
  -H "Authorization: Bearer $GITHUB_TOKEN" \
  -H "Accept: application/vnd.github+json" \
  "https://api.github.com/repos/owner/repo/pulls/123/reviews" \
  -d '{
    "body": "一括返信レビュー",
    "event": "COMMENT",
    "comments": [
      {"path": "file.js", "line": 10, "body": "返信1"},
      {"path": "file.py", "line": 25, "body": "返信2"}
    ]
  }'
```
